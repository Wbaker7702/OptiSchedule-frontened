# Security Policy

## Supported Versions

This project is currently in a pre-release stage. As such, there are no versions that are formally supported with security updates. This policy will be updated as the project matures and official releases are made.

## Reporting a Vulnerability

We take security vulnerabilities seriously. If you discover a security issue, please report it to us by emailing `security@example.com`. We will do our best to respond within 48 hours of receiving your report.
